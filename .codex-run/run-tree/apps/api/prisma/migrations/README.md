# Prisma Migrations

This directory contains the committed Prisma migrations for the Phase 0
foundation schema.

The current initial migration is generated by Prisma in timestamped directory
form and includes the foundation health metadata, authentication baseline,
security audit records, operational events, and background job metadata required
by `specs/001-foundation-architecture/tasks.md`.

Use `pnpm db:migrate` from the repository root to apply migrations through the
workspace script.
